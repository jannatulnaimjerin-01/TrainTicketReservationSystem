# Train Ticket Reservation System

## Course Information
- Course Name: Object Oriented Programming Language Lab
- Language: C#

## Project Description
The Train Ticket Reservation System is developed using C# as part of the
Object Oriented Programming Lab course.

This system allows users to:
- View available trains
- Reserve train tickets
- Store passenger information
- Generate tickets

## OOP Concepts and Features Used
The project is implemented using the following Object Oriented Programming
concepts and features:

- Classes
- Constructors
- Constructor Overloading
- Copy Constructors
- Function Overloading
- Operator Overloading
- Static Methods and Static Fields
- Inheritance
- Interfaces
- Virtual Functions
- Abstract Classes and Abstract Methods
